# Calculadora

Projeto de uma calculadora simples e responsiva desenvolvida com HTML, CSS e JavaScript.

## Funcionalidades

- Soma
- Subtração
- Multiplicação
- Divisão
- Limpar visor
- Interface responsiva para celular e computador

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript

## Estrutura do projeto

- index.html
- style.css
- script.js

## Como executar

1. Baixe os arquivos
2. Abra a pasta do projeto
3. Execute o arquivo `index.html` no navegador

## Objetivo

Projeto criado para praticar lógica de programação, manipulação do DOM e integração entre HTML, CSS e JavaScript.
